## Cold Chain Logistics System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/1e06ceb041b54745b7b345274fdc1ff8/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Courier, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Announcements
5. Goods
6. Message Board
7. Branch Information
8. Personal Center
9. Delivery Address
10. Goods Orders


**Courier Functions:**
1. Login
2. Register
3. Change Password
4. Personal Information
5. Announcement Management
6. Order Assignment Management
7. Message Board Management
8. Branch Information Management


**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Administrator Management
4. Courier Management
5. User Management
6. Announcement Management
7. Goods Management
8. Goods Order Management
9. Order Assignment Management
10. Message Board Management
11. Branch Information Management
12. Basic Data Management (Announcement Type Management, Goods Type Management, Order Type Management, Branch Type Management)
13. Carousel Management
